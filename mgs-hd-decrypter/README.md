# mgs-hd-decrypter

A tool to decrypt Metal Gear Solid 2 (Sons of Liberty) and Metal Gear Solid 3 (Snake Eater) PS3 save-games

Note: this tool also updates the custom integrity checksums.

```
USAGE: ./mgs-hd-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
